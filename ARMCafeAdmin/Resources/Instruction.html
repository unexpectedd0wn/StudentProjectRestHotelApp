Запуск программы.

Для начала работы с АРМ «Администратор кафе» запустите ярлык «АРМ администратор» расположенный на рабочем столе.

Для того чтобы выполнить авторизацию пользователя необходимо ввести учетные данные пользователя, если данные верны – то Вам будут доступны разделы:

-          Главная;

-          Бронирование;

-          Банкет;

-          Меню банкета;

-          Официанты;

-          Отчеты;

-          О программе;

-          Выход;

 

Главная

На «Главной» форме доступны информационные плитки:

-          Количество бронирований столов на сегодня;

-          Количество банкетов на сегодня;

-          Количество официантов, работающих сегодня;

 

Бронирование

В пользовательском меню необходимо выбрать пункт: Бронирование.

Для добавления нового бронирования, в разделе «Детали бронирования» необходимо заполнить все поля раздела «Детали бронирования» и нажать кнопку «Сохранить». Для редактирования, необходимо кликнуть двойным щелчком мыши на выбранном бронировании из «Списка бронирований». В разделе «Детали бронирования», загрузится детальная информация о бронировании. Далее можно отредактировать данные и нажать кнопку «Обновить».

Для поиска необходимо в верхней части формы ввести поисковый запрос и нажать кнопку «Поиск». Для отчистки результатов поиска и текстового поля поиска необходимо нажать кнопку «Отчистить».

При нажатии кнопки «Отмена» сбрасываются все введенные значения в разделе «Детали бронирования».

Для экспорта, в верхней части формы необходимо нажать кнопку «Экспорт». Файл будет сохранен по адресу: C:\Users\ {current user}\Documents\АРМ Администратор кафе/

 

Банкет

В пользовательском меню необходимо выбрать пункт: Банкет.

Для регистрации нового Банкета, в разделе «Основная информация» необходимо заполнить все поля и нажать кнопку «Сохранить». Для редактирования, необходимо кликнуть двойным щелчком мыши на выбранном банкете из «Список банкетов». В разделе «Основная информация», загрузится детальная информация о банкете. Далее можно отредактировать данные и нажать кнопку «Обновить».

При нажатии кнопки «Отмена» сбрасываются все введенные значения в разделе «Детали бронирования».

Для экспорта, в верхней части формы необходимо нажать кнопку «Экспорт». Файл будет сохранен по адресу: C:\Users\ {current user}\Documents\АРМ Администратор кафе/

 

Для добавления позиций меню в банкет, необходимо кликнуть двойным щелчком мыши на выбранном банкете из «Список банкетов». Далее, в нижней части формы в разделе «Меню» выбрать позицию меню и указать кол-во и нажать кнопку «Добавить». Для Редактирования или удаления позиции меню, необходимо кликнуть двойным щелчком мыши на выбранной позиции меню, внести изменения и нажать кнопку «Обновить» или «Удалить».

Для поиска необходимо в верхней части формы ввести поисковый запрос и нажать кнопку «Поиск». Для отчистки результатов поиска и текстового поля поиска необходимо нажать кнопку «Отчистить».

Для экспорта меню, необходимо выбрать Банкет, двойным щелчком мыши и в нижней части формы в разделе «Меню» необходимо нажать кнопку «Экспорт». Файл будет сохранен по адресу: C:\Users\ {current user}\Documents\АРМ Администратор кафе/

 

Меню банкета

В пользовательском меню необходимо выбрать пункт: Меню банкета.

Для добавления новой позиции меню, в разделе «Детали блюда» необходимо заполнить все поля раздела и нажать кнопку «Сохранить». Для редактирования, необходимо кликнуть двойным щелчком мыши на выбранной позиции меню из «Список блюд». В разделе «Детали блюда», загрузится детальная информация о позиции меню. Далее можно отредактировать данные и нажать кнопку «Обновить».

Для поиска необходимо в верхней части формы ввести поисковый запрос и нажать кнопку «Поиск». Для отчистки результатов поиска и текстового поля поиска необходимо нажать кнопку «Отчистить».

При нажатии кнопки «Отмена» сбрасываются все введенные значения в разделе «Детали блюда».

Для экспорта, в верхней части формы необходимо нажать кнопку «Экспорт». Файл будет сохранен по адресу: C:\Users\ {current user}\Documents\АРМ Администратор кафе/

 

Официанты

В пользовательском меню необходимо выбрать пункт: Официанты.

Для добавления нового официанта в разделе «Детали официанта» необходимо заполнить все поля раздела и нажать кнопку «Сохранить». Для редактирования, необходимо кликнуть двойным щелчком мыши на выбранном официанте из «Список официантов». В разделе «Детали официанта», загрузится детальная информация о официанте. Далее можно отредактировать данные и нажать кнопку «Обновить» или нажать кнопку «Удалить».

Для проставления смены, необходимо в разделе «Добавить смену» выбрать из выпадающего списка фамилию официанта и выбрать дату в календаре. Далее нажать кнопку «Добавить». Для редактирования «смены», необходимо двойным щелчком выбрать смену, и отредактировать введенные данные в разделе «Добавить смену» или нажать кнопку «Удалить» в верхней части «Список смен».

Для экспорта, в верхней части раздела «Список смен» необходимо нажать кнопку «Экспорт». Файл будет сохранен по адресу: C:\Users\ {current user}\Documents\АРМ Администратор кафе/

Выход

Для закрытия приложения в пользовательском меню необходимо выбрать пункт: Выход или нажать на X в верхнем правом углу формы. После появится окно подтверждения закрытия и нажать ОК.
